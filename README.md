# ⛏️ Análisis Geoquímico de Rocas Ígneas

Este proyecto desarrolla un análisis geoquímico utilizando **Python y Pandas** para estudiar la composición de rocas ígneas, con aplicaciones en la industria minera.

---

## 🚀 Funcionalidades

- 🧹 Limpieza de datos geoquímicos  
- 📊 Análisis estadístico  
- 🔗 Matriz de correlación  
- 📈 Visualización de datos  
- 🌍 Análisis geoespacial  
- 💻 Aplicación web interactiva con Streamlit  

---

## 🛠️ Tecnologías utilizadas

- 🐍 Python  
- 🧮 Pandas  
- 📊 Matplotlib  
- 🎨 Seaborn  
- 🌐 Streamlit
